Введение в HTML и CSS

Основы создания веб-страниц с использованием HTML и CSS

Что такое HTML?

HyperText Markup Language — это стандартный язык разметки документов для создания веб-страниц. Он используется для структурирования содержимого на веб-страницах и определения их элементов, таких как заголовки, абзацы, ссылки, изображения и другие типы контента.

Основные теги HTML

  • <html> — корневой элемент
  • <head> — метаинформация, контейнер для таких элементов как title, style, meta, link, script
  • <title> — заголовок страницы, отображающийся на верхней панели браузера
  • <body> — основное содержимое, включает весь контент который видит пользователь
  • <h1>-<h6> — заголовки, от наиболее значимого до наимее, должны отличаться по размеру шрифта
  • <p> — параграф, текст который содержит более одной строки
  • <a> — ссылка
  • <img> — изображение

Что такое CSS?

Cascading Style Sheets — это язык иерархических таблиц стилей, используемый для представления внешнего вида документа, написанного на HTML или XML, включая различные языки XML, такие как SVG и XHTML. CSS описывает, каким образом элемент должен отображаться на экране.

Управляет цветами, шрифтами, отступами наших тегов.

Основные свойства CSS

  • color — цвет текста
  • background-color — цвет фона
  • font-size — размер шрифта
  • margin — внешние отступы
  • padding — внутренние отступы
  • border — граница элемента

Пример использования HTML и CSS


        <!DOCTYPE html>
        <html>
        <head>
          <style>
            body {
              background-color: #f0f0f0;
              font-family: Arial, sans-serif;
            }
            h1 {
              color: #333;
              text-align: center;
            }
            p {
              color: #666;
              font-size: 14px;
              line-height: 1.6;
            }
          </style>
        </head>
        <body>
          <h1>Привет, мир!</h1>
          <p>Это пример простой HTML-страницы с использованием CSS.</p>
        </body>
        </html>

Заключение

HTML и CSS — основа веб-разработки. Освоение этих технологий открывает двери в мир создания веб-сайтов.
